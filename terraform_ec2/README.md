# Terraform AWS EC2 Infrastructure Project

## Overview

This project demonstrates Infrastructure as Code (IaC) using Terraform
to provision AWS resources including VPC, subnet, security group, and
EC2 instance.

## Technologies

-   Terraform
-   AWS EC2, VPC

## Setup

1.  terraform init
2.  terraform plan
3.  terraform apply

## Resources Created

-   VPC
-   Subnet
-   Internet Gateway
-   Route Table
-   Security Group (SSH)
-   EC2 Instance

## Output

-   Public IP for SSH access
